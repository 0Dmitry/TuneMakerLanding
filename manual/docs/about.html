    What is Tune Maker?

    Tune Maker is a music composition tool for iOS. It helps you to understand the basic components of music and to quickly begin creating your own compositions. It can be used by musicians of all levels. Beginners can start using basic chords, rhythms and melodies. Professional musicians will appreciate the ability to construct very advanced chords, using non-standard time signatures, adding multiple melody lines, MIDI export, recording chords and notes from a MIDI keyboard and many more.
