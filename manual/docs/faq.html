    FAQ

    How do I copy note events from one track to the other?

    Select events you want to copy. Tap Copy button from the appeared toolbar. Then tap on the track selection button(colored square at bottom right) and select the track you want to copy to.

    You can copy note events from one track to another only within the same track group. For example you can only copy events from one melody track to another melody track. You can not move melody events to an arpeggio track or a drum track.

    How do I export my project as a MIDI file?

    Navigate to your project file in the document browser, long-press on the file, select Export as MIDI.
