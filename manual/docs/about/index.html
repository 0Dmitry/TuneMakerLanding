    What is Tune Maker?

    Tune Maker is a music composition tool for iOS. It is a universal app, so it can run on both iPhone and iPad. It requires iOS 11 or later.

    Tune Maker is a document-based application, which means you can store your project files(your tunes) in a way that works best for you. You can store projects locally on your device, on your iCloud folder, or in any available third-party cloud service like DropBox or GoogleDrive.

    Tune Maker supports USB and Bluetooth MIDI controllers. So you can connect your MIDI keyboard and record notes and chords.

    Tune Maker comes with a basic set of built-in instruments and drum kits. You can also connect it to any CoreMIDI compatible app or plug in any Audio Unit Extension instrument available on the App Store.

    Tune Maker allows yout to export your project as a MIDI file if you want to continue working on it in a DAW like Ableton or GarageBand
